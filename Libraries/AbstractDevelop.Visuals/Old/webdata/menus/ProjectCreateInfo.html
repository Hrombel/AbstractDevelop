Создание проекта

Выбор абстрактного вычислителя

При запуске программы окно создания проекта открывается автоматически. Оно предоставляет выбор абстрактного вычислителя, под который создается проект. При нажатии на соответствующие элементы меню справа будет отображаться вспомогательная информация о конкретном абстрактном вычислителе. Когда выбор сделан, необходимо воспользоваться строкой ввода названия проекта снизу. Название проекта может содержать только буквы латинского алфавита и/или цифры. Справа от строки ввода находится флаг, позволяющий определить, необходимо ли создать одноименную папку для проекта в директории, которая будет впоследствии выбрана. Далее, необходимо нажать на кнопку создать и выбрать нужную директорию.
