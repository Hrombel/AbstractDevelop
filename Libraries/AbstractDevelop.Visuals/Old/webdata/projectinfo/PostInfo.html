Машина Поста

Принцип работы

Машина Поста состоит из каретки (или считывающей и записывающей головки) и разбитой на ячейки бесконечной в обе стороны ленты. Каждая ячейка ленты может находиться в 2 состояниях — быть либо пустой, либо отмеченной. За такт работы машины каретка может сдвинуться на одну позицию влево или вправо, считать, изменить символ в своей текущей позиции. Работа машины Поста определяется программой, состоящей из конечного числа строк. Для работы машины нужно задать программу и её начальное состояние (то есть состояние ленты и позицию каретки). Кареткой управляет программа, состоящая из пронумерованных строк команд. Если в команде переход не указан, то переход происходит на следующую строку. Каждая команда имеет следующий синтаксис:
p // Установить метку в ячейку, напротив которой находится каретка.
e // Стереть метку из ячейки, напротив которой находится каретка.
r // Переместить головку вправо.
l // Переместить головку влево.
j a b // Условный переход: если напротив текущей ячейки находится метка, переход к команде a, иначе - к b.
s // Остановка выполнения программы.
Следует отметить, что после объявления каждой команды, кроме последних двух, допустимо указывать либо опускать номер команды, к выполнению которой совершается переход. Если в ходе выполнения программы произведен переход к команде с неизвестным номером, выполнение программы прекращается. При осуществлении попытки установки метки в ячейку, имеющую метку, либо при попытке стирания несуществующей метки выполнение программы прекращается с ошибкой.
